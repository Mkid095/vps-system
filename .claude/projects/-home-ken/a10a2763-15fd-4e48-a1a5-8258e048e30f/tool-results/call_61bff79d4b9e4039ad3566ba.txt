Maven Flow - Autonomous Development Orchestrator

Usage:
  flow start [--dry-run] [max-iterations]  - Start flow (default: 100 iterations)
  flow status                                  - Show current status
  flow continue                                - Continue from previous session
  flow reset                                   - Reset session state
  flow help                                    - Show this help

Options:
  --dry-run    Show what would happen without making changes
